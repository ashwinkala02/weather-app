🌦️ Weather App

A modern and responsive **Weather Application** built using **HTML, CSS, and JavaScript**, providing real-time weather updates and forecasts using the OpenWeather API.

---

📌 Features

✔️ Search weather by city name
✔️ Get weather using current location (Geolocation)
✔️ 5-day weather forecast
✔️ Popular cities weather dashboard
✔️ Add and view favorite cities (LocalStorage)
✔️ Clean and responsive UI

⚙️ How to Run Locally

1. Clone the repository

```
git clone https://github.com/ashwinkala02/weather-app.git
```

2. Open the project folder

3. Run `index.html` in your browser
